Задание на разработку приложения
Приложение пишем на C#
Для работы под Win10, Win11
Дизайн в стиле Win11.

Приложение имеет графический интерфейс.
Пользователь открывает папку с видеофайлами через пункт меню.
Видеофайлы могут быть _.mp4, _.avi
Приложение создает в папке файл "Anonce.md"

Описание интерфейса:
Видео выводится в основное окно приложения.
В надписях на кнопках присутствуют названия дублирующих их клавиш.
Основное окно приложения делится на 3 части:
слева большая часть окна -для показа видеоролика.
справа 20% ширины окна и во всю высоту - область просмотра содержимого папки.
снизу область для кнопок - во всю ширину окна.
Интерфейс содержит кнопки управления скоростью воспроизведения - Кнопками 1,2,3,4 пользователь может управлять скоростью воспроизведения. 1 - нормальная скорость, 2 - скорость x2 и т.д.
Кнопка "START" По кнопке START открывается первый видеофайл. При достижении конца видеофайла, ждать команды пользователя.
Кнопка "NEXT" - по ней происходит переход к следующему видеоролику.
Кнопка "BEGIN" - при нажатии на нее или при нажатии на клавишу "q" или "Q" засекается временная позиция видеоролика Time1.
Кнопка "END" - при нажатии на нее или на клавишу "w" или "W" засекается вторая временная позиция видеоролика Time2.
Навигация по названиям видеофайлов происходит либо в автоматическом режиме, либо пользователь может мышью указать на нужный файл, тогда приложение начнет его обработку и двалее будет переходить к следующим за ним файлам.

Как только обе временные метки готовы, в файл "Anonce.md" заностится строка:
<Имя видеофайла> "Start "Time1, "Stop "Time2.
После каждой записи, файл сохраняется.
При закрытии приложения, если не все файлы в папке обработаны, или просматриваемый файл просмотрен не до конца, создается файл "Continue.md" в который заностися имя файла с которого следует продолжить выполнение и время видеоролика, на котором остановился просмотр, если он не был закончен.
Ели все файлы в папке обработаны, то файл "Continue.md" не создается.
При закрытии приложения, если не все файлы обработаны, выпадает окно с предупреждением что не все файлы обработаны.

Если пользователь пытается обработать файл повторно, приложение не даст этого сделать, пока из файла "Anonce.md" не будут убраны строки относящиеся к этому файлу.

При старте приложение ищет файл "Continue.md" и продолжает работу с указанного в нем места.
